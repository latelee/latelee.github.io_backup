Linux常用命令记录
<!- more -->

将当前目录所有文件的intMAX字符串替换为INTd_MAX：
```
sed -i 's/intMAX/INT_MAX /g' `grep MAX_INt ./ -rl` 
```
注：可以修改grep查找路径来指定目录。


仅查看当前目录所有.cpp和.c文件：
```
find ./ -maxdepth 1 -name '*.cpp' -or -name '*.c'
```
注：使用-maxdepth 1指定搜索的目录深度。去掉的话，则是递归搜索所有子目录。

去掉行尾多余空格
```
sed -i "s/[ ]*$//g" test.cpp
```
将tab键替换为4个空格
sed -i "s/\t/    /g" test.cpp
